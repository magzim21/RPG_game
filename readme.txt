  Общее описание: прототип консольной рпг игрушки, где от первого лица, командами, управляется сущность типа "игрок". Игрок находится и передвигается между локациями, назовем их "комнатами". Комнаты будут находиться друг по отношению к другу по типу (север, запад, юг, восток, вверх, вниз). Комната, в первом приближении, должна иметь название, описание, выходы. "Игрок" всегда знает в какой комнате он находится. В целом, требование к тому, должна ли знать комната про игрока внутри себя в данном задании необязательно.

Пользовательский ввод будет передаваться методу run, для обработки;
Зарегистрировать команды: смотреть для метода look, восток, север, запад, юг, для метода move;
